# Bu sistem 2 ayrı pencereden oluşmaktadır;

* GİRİŞ PENCERESİ: Kullanıcının giriş yaptığı penceredir.


* KULLANICI PENCERESİ: Kullanıcının dilediği kişini T.C'sini girerek bilgilerine ulaştığı penceredir.
